# QR Kod Oluşturucu ve Okuyucu

## Proje Hakkında
Bu proje, kullanıcıların metinleri QR koda dönüştürmesini ve QR kodları okumasını sağlar.

## Özellikler
- Metni QR koda dönüştürme
- QR kod okutma
- Kullanıcı dostu arayüz

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript
